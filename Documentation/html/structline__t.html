Detailed Description

Заголовочный файл с прототипами функций из line_t.c.

Хранит указатель на си-строку и длину этой строки

В этой программе массивы структур line_t заканчивается нулевой структурой line_t. Нулевая структура - такая, что ptr = NULL, len = -1; Нулевая структура != структура с пустой строкой. Пустая строка содержит единственный символ \0, ptr != NULL и len = 0, а не -1. Такое оформление массива позволяет не передавать длину массива. Функция lineIsNull() проверяет, является ли объект структуры нулевым.
